TLDR: An "A" means you have a right number in the right spot. A "B" means you have a right number in the wrong spot.

Goal

The goal of Moo is to guess a random number the game has generated. This number is defaulted to four digits, but you can adjust this in the Settings tab.

Results

Each time you make a guess, the game will give you a result. If you guess it correctly, it will say "You win!". If you guess it incorrectly, it will give you a result using the letters "A" and "B". This result is a hint to guide you to the correct answer.

An "A" means you have a right number in the right spot. A "B" means you have a right number in the wrong spot.

Examples

If the number is "2014" and you guess "1234", the result will be "1A 2B". The "4" that you guessed is a right number in the right spot. The "1" and "2" are right numbers, but in the wrong spots. The "3" isn't in the answer so no result is given for it. Note that answers can start with zero as well.

More examples

If the answer is 7890:

Guess: 1234 --> Response: 0A 0B
Guess: 5678 --> Response: 0A 2B
Guess: 7809 --> Response: 2A 2B
Guess: 8888 --> Response: 1A 3B

Repeat that!

In the settings, you can also allow repeated numbers, such as "1122". The rules for the result are the same. Below are some examples for your reference. Say the answer is "5567":

Guess: 1155 --> Response: 0A 2B
Guess: 2345 --> Response: 0A 1B
Guess: 4550 --> Response: 1A 1B
Guess: 5555 --> Response: 2A 2B

Source

You can also read the rules on Wikipedia. The game is called "Bulls and cows", and one of the first computer implementations of it was called "moo", hence the name of this game.

No cows were harmed in the production of this app.
